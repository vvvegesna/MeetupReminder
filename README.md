# MeetupReminder
100 Days of SwiftUI, Day 77 and 78 challenges

Requires XCode 12 to execute on simulator. Note, don't forget to simulate location while using simulator.

Apart from the given discription by https://www.hackingwithswift.com/users/twostraws, user has facility to enter location either by manually adding a pin or by
directly tapping on currentLocation icon
